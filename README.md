# Personal Portfolio Website

This project is a Personal Portfolio Website developed using HTML5 and CSS3. It showcases my profile, education, skills, projects, and contact information.

## Features

* Professional portfolio design
* About Me section
* Education details
* Skills showcase
* Projects section
* Contact information

## Technologies Used

* HTML5
* CSS3

## Learning Outcomes

Through this project, I improved my web development, UI design, responsive layout, and portfolio-building skills.
